# Commands to Push to GitHub

# After creating repo on GitHub, run these commands:

cd C:/Users/georg/multi-agent-pm

# Add GitHub as remote
git remote add origin https://github.com/shawngeorgie06/multi-agent-pm.git

# Push to GitHub (creates main branch)
git branch -M main
git push -u origin main

# Verify it worked
# Go to: https://github.com/shawngeorgie06/multi-agent-pm

# ===============================================
# IMPORTANT: Before Pushing
# ===============================================
# 1. Your backend/.env file is gitignored (safe)
# 2. Only backend/.env.example will be pushed
# 3. Your real API key stays local
#
# If you want to use your real API key locally:
# Edit backend/.env and replace placeholder with:
# GEMINI_API_KEY=your_real_key_here
